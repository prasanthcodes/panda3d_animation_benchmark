# panda3d animation benchmark
### Run this code and set the number of objects to find out the performance of animations in your machine.

## shortcuts

### mouse controls
right click and drag to view 3d scene

### keyboard shortcuts

w - move forward <br/>
s - move backward <br/>
a - move left <br/>
d - move right <br/> <br/>

b - gravity on <br/>
m - to hide all GUI <br/> <br/>
